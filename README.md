Texibition Healthcare is a high-performance, full-stack digital infrastructure designed to provide instant, reliable medical services. By integrating real-time ambulance tracking, specialist consultations, and pharmacy logistics into a single, intuitive interface, we are redefining the patient experience.

---

## 🚀 Key Features

### 🚑 Emergency Response
*   **Smart Ambulance Booking**: One-tap emergency requests with live GPS tracking and hospital routing.
*   **Critical Helpline**: 24/7 direct access to emergency medical dispatchers.

### 👨‍⚕️ Clinical Services
*   **Specialist Discovery**: Find and filter doctors by expertise, rating, and proximity.
*   **Virtual Ward**: HD-video consultations with integrated medical report sharing.

### 💊 Logistics & Diagnostics
*   **Digital Pharmacy**: Full-inventory medical store with prescription-based ordering and home delivery.
*   **Home Lab Service**: Professional sample collection for diagnostic tests with digital report delivery.

### 🏥 Hospital Network
*   **Live Hospital Map**: Interactive Leaflet-based map showing verified facilities and emergency status.
*   **Notice Board**: Real-time hospital announcements, surgery schedules, and blood donation alerts.

---

## 🛠️ Comprehensive Tech Stack

### **Frontend & UI**
- **Next.js 16 (App Router)**: Hybrid rendering for optimized SEO and blistering performance.
- **React 19**: Utilizing the latest concurrent features for a fluid user experience.
- **Tailwind CSS 4**: A utility-first styling engine for rapid, responsive UI development.
- **Framer Motion**: State-of-the-art physics-based animations for intuitive transitions.
- **Shadcn UI**: Accessible, high-fidelity UI primitives.

### **Backend & Infrastructure**
- **Firebase Auth**: Enterprise-grade identity management and secure user sessions.
- **Cloud Firestore**: Real-time NoSQL database for instant sync across all user devices.
- **Firebase Storage**: Scalable cloud storage for patient reports and prescriptions.
- **Netlify Edge**: Global CDN deployment for ultra-low latency.

### **Geospatial & Utilities**
- **Leaflet & React Leaflet**: High-performance interactive mapping for ambulance and hospital tracking.
- **Lucide React**: Clean, consistent iconography.
- **jsPDF & html2canvas**: Client-side medical report generation and export.

## 👤 User Use Cases

### 🚨 Scenario 1: The Emergency
*   **User**: A bystander witnessing an accident.
*   **Action**: Opens Texibition Healthcare, taps "Emergency Ambulance".
*   **Outcome**: Location is automatically detected, nearest ambulance is dispatched, and the user tracks it in real-time.

### 💼 Scenario 2: The Busy Professional
*   **User**: Someone feeling unwell during a workday.
*   **Action**: Books a "Virtual Consultation" for their lunch break.
*   **Outcome**: Connects with a doctor via secure video, receives a digital prescription, and orders medicines for home delivery within the same app.

### 🧪 Scenario 3: Routine Diagnostics
*   **User**: A patient needing a monthly blood sugar check.
*   **Action**: Schedules a "Lab Test" for home collection at 7:00 AM.
*   **Outcome**: A professional collector arrives on time; results are synced directly to the user's dashboard by evening.

### 🏥 Scenario 4: Hospital Discovery
*   **User**: Someone new to a city looking for a nearby cardiologist.
*   **Action**: Uses the "Hospital Finder" map.
*   **Outcome**: Filters by "Cardiology", checks verified ratings, and gets instant driving directions.

## 📦 Installation & Setup

1.  **Clone & Navigate**
    ```bash
    git clone https://github.com/your-username/texibition-healthcare.git
    cd healthcare-platform
    ```

2.  **Environment Configuration**
    Create `.env.local`:
    ```env
    NEXT_PUBLIC_FIREBASE_API_KEY=your_key
    NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
    # ... other firebase vars
    ```

3.  **Start Development**
    ```bash
    npm install && npm run dev
    ```
